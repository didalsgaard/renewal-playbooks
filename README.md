
# Renewal Playbook

This repository captures my approach to managing and closing customer renewals in B2B SaaS.

## What this is
- A practical, experience-driven guide
- Focused on risk reduction, value reinforcement, and timing

## What this is not
- A generic CS textbook
- A one-size-fits-all framework

## Topics covered
- Renewal timelines
- Risk signals
- Stakeholder alignment
- Metrics and health scoring
